Statistical Analysis with SPSS

Course Objectives

  • Understand SPSS functionalities and interface.
  • Perform descriptive and inferential statistical analyses.
  • Interpret SPSS outputs effectively.
  • Use SPSS for hypothesis testing, regression analysis, and advanced data modeling.
  • Create meaningful visualizations and reports.

Course Structure

Chapter 1: Introduction to SPSS

  • Installing SPSS and Overview of Interface
  • Data Entry and File Management
  • Importing and Exporting Data (Excel, CSV, etc.)

Activity: Enter survey data into SPSS.


Chapter 2: Data Cleaning and Management

  • Handling Missing Data
  • Recoding Variables and Computing New Variables
  • Sorting, Filtering, and Merging Datasets

Activity: Clean a messy dataset and prepare it for analysis.


Chapter 3: Descriptive Statistics

  • Measures of Central Tendency (Mean, Median, Mode)
  • Measures of Dispersion (Range, Variance, Standard Deviation)
  • Frequency Distributions and Crosstabs

Activity: Analyze and interpret data from a sales dataset.


Chapter 4: Data Visualization

  • Creating Bar Charts, Pie Charts, and Histograms
  • Boxplots and Scatterplots
  • Customizing Charts for Reports

Activity: Visualize data from a customer feedback survey.


Chapter 5: Inferential Statistics

  • Hypothesis Testing (t-tests, Chi-square tests)
  • ANOVA (One-way and Two-way)
  • Correlation Analysis

Activity: Test hypotheses using provided datasets.


Chapter 6: Regression Analysis

  • Linear Regression
  • Logistic Regression
  • Interpreting Regression Outputs

Activity: Build a regression model to predict sales.


Chapter 7: Advanced Statistical Methods

  • Factor Analysis
  • Reliability Analysis (Cronbach's Alpha)
  • Cluster Analysis

Activity: Perform factor analysis on a psychological dataset.


Chapter 8: Reporting and Automation

  • Creating SPSS Syntax for Reproducibility
  • Exporting Results to Word/Excel
  • Automating Reports

Activity: Generate a complete report from analysis.


Interactive Features

  • Hands-on Exercises: Work with real-world datasets (e.g., healthcare, marketing, academic performance).
  • Quizzes and Assignments: Test understanding after each module.
  • Live Sessions: Weekly live Q&A or practice sessions with an instructor.
  • Case Studies: Analyze real-world scenarios, such as clinical trial data or market research.
  • Community Discussions: Forums for learners to share insights and solve problems collaboratively.
